Tipos primitivos:

  • String: Tipo primitivo para textos.
  • Number: Tipo primitivo para números.
  • Boolean: Tipo primitivo para os valores: True ou False.
  • Undefined: Tipo primitivo para indicar dados(variáveis etc) não existentes
  • Null: Tipo primitivo para dados(variáveis) vazias.
  • Symbol: Tipo primitivo para símbolos.

Outros tipos:

  • Object: Tipo para objetos.
  • Array: Tipo para array.

Cadeia de caracteres:

  • Para definir um tipo de dado do tipo string podemos usar: " "=> aspas duplas; ' ' => aspas simples.
  • Também pode-se utilizar ` ` => crase para template strings.
  • Propriedades e Métodos das strings:
    • Tamanho de uma string: .lenght
      Ex:
      • let minhaString = "ABCDEF";
        let tamanho = minhaString.lenght;
    • Extrai uma parte de uma string e colocar em uma nova string: .slice(start, end)
      Ex:
      • let str = "Apple, Banana, Kiwi";
        let part = str.slice(-12, -6); // part = 'Banana'
    • Extrai uma parte de uma string, similar ao método anterior, não aceita índices negativos: .substring(start, end)
      Ex:
      • let str = "Apple, Banana, Kiwi";
        let part = str.substring(7, 13); // part = 'Banana'
    • Extrai uma parte de uma string, similar ao método ..slice(), segundo parâmetro é o tamanho da nova string: .substr(start, lenght)
      Ex:
      • let str = "Apple, Banana, Kiwi";
        let part = str.sub(7, 6); // part = 'Banana'
    • Substituir o valor de uma string: .replace("palavra", "outraPalavra")
      Ex:
      • let text = "Please visit Microsoft!";
        let newText = text.replace("Microsoft", "W3Schools"); // newText = 'Please visit W3Schools!'
    • Converter toda a string para letras maiúsculas: .toUpperCase()
      Ex:
      • let t1 = "Hello, World!";
        let t2 = t1.toUpperCase(); // t2 = 'HELLO, WORLD!'
    • Converter toda a string para letras minúsculas: .toLowerCase()
      Ex:
      • let t1 = "Hello, World!";
        let t2 = t1.toLowerCase(); // t2 = 'hello, world!'
    • Retornar um caractere de uma posição da string: .charAt(posicao)
      Ex:
      • let t1 = "HELLOWORLD";
        let char = t1.charAt(0); // char = H
    • Retornar o código ASCII de uma posição da string: .charCodeAt(posicao)
      Ex:
      • let t1 = "HELLOWORLD";
        let char = t1.charCodeAt(0); // char = 72
    • Retornar o caractere de uma posição da string: propertyaccess[]
      Ex:
      • let t1 = "HELLOWORLD";
        let char = t1[0]; // char = H
    • Retornar o caractere de uma posição da string: propertyaccess[]
      Ex:
      • let t1 = "HELLOWORLD";
        let char = t1[0]; // char = H
    • Convertendo uma string em array: .split(ondeSeparar)
      Ex:
      • let t1 = "HELLO WORLD";
        let t2 = t1.split(",") // ",", "|"; // t2 = ['Hello', 'World!']
    • Buscando um valor em uma string: .indexOf() / Aceita-se um segundo parâmetro para definir o começo da busca, retorna o index
      Ex:
      • let t1 = "Please locate where 'locate' occours!";
        let t2 = t1.indexOf("locate"); // t2 = 7
    • Buscando um valor em uma string: .lastIndexOf() / retorna o último index da palavra.
      Ex:
      • let t1 = "Please locate where 'locate' occours!";
        let t2 = t1.lastIndexOf("locate"); // t2 = 21
    • Buscando um valor em uma string: .search() / não aceita segundo parâmetro, aceita expressão regular.
      Ex:
      • let t1 = "Please locate where 'locate' occours!";
        let t2 = t1.search("locate"); // t2 = 7
    • Para saber se uma string inicia com uma palavra: .startsWith() / retorna true ou false.
      Ex:
      • let t1 = "Please locate where 'locate' occours!";
        let t2 = t1.startsWith("locate"); // t2 = false
    • Para saber se uma string termina com uma palavra: .endsWith() / retorna true ou false.
      Ex:
      • let t1 = "Please locate where 'locate' occours!";
        let t2 = t1.endsWith("locate"); // t2 = false

Números

  • O tipo Number do JavaScript só tem dois tipos: com ou sem casas decimais.
  • O número máximo de casas decimais é 17.
  • Classificação:
    • NaN (Not a Number): indica que um número está incorreto.
    • Infinity(ou -Infinity): descreve um número que tem mais casas decimais do que o que é possível expor.
  • Métodos e funções para números:
    • Função que retorna o valor absoluto de um número: Math.abs()
      Ex:
      • console.log(Math.abs(-5.2)); // 5.2
        console.log(Math.abs(5.2)); // 5.2
    • Função que arrendondar um número (float) para cima: Math.ceil()
      Ex:
      • console.log(Math.ceil(0.8)); // 1
        console.log(Math.ceil(-0.8)); // 0
    • Função que arrendondar um número (float) para baixo: Math.floor()
      Ex:
      • console.log(Math.floor(0.8)); // 0
        console.log(Math.floor(-0.8)); // -1
    • Função que arrendonda um float para o número inteiro mais próximo: Math.round()
      Ex:
      • console.log(Math.round(0.8)); // 1
        console.log(Math.round(-0.8)); // -1
        console.log(Math.round(4.05)); // 4
    • Função que corta as casas decimais de um float: Math.trunc()
      Ex:
      • console.log(Math.trunc(0.8)); // 0
        console.log(Math.trunc(-0.8)); // 0
        console.log(Math.trunc(4.05)); // 4
    • Função que retorna o maior número de uma coleção: Math.max()
      Ex:
      • console.log(Math.max(2, 0.8, 5, -15)); // 5
    • Função que retorna o menor número de uma coleção: Math.min()
      Ex:
      • console.log(Math.min(2, 0.8, 5, -15)); // -15
    • Função que retorna o resultado de uma potência: Math.pow()// equivale no EcmaScript 6 => (**)
      Ex:
      • console.log(Math.pow(2, 3)); // 8
        console.log(2**3); // 8
    • Função que retorna o resultado de uma raiz quadrada: Math.sqrt()// equivale no EcmaScript 6 => (** (1/2))
      Ex:
      • console.log(Math.sqrt(9)); // 3
        console.log(9 ** (1/2)); // 3
    • Função que retorna o resultado de uma raiz cúbica: Math.cbrt()// equivale no EcmaScript 6 => (** (1/3))
      Ex:
      • console.log(Math.cbrt(27)); // 3
        console.log(27 ** (1/3)); // 3
    • Função que retorna um npumero entre 0 e 1: Math.random()
      Ex:
      • console.log(Math.random()); // 0.8365673319152338

Objetos:

  • O tipo Object do JavaScript é uma coleção dinâmica que apresenta o sistema de chave e valor. // qualquer tipo de dado pode ser utilizado.
  • É possível adicionar ou remover propriedades a qualquer momento.
  • Ex:
                                
                        let pessoa = {
                            nome: "João",
                            idade: 20,
                            telefone: null,
                            endereco: {
                                logradouro: "Rua Brasil",
                                numero: 1301,
                                bairro: "Centro"
                            }
                        }
    console.log(pessoa.nome);// "João"
    console.log(pessoa["idade"]); //20
    console.log(pessoa.telefone);// null
    console.log(pessoa.endereco.logradouro); // "Rua Brasil"
    console.log(pessoa["endereco"]["numero"]); //1301
    console.log(pessoa.endereco.bairro); //"Centro"
    console.log(pessoa.peso); // undefined
  • Métodos para trabalhar com objetos: códico acima.
    • Remoção de um atributo de um objeto: delete
      Ex:
      • delete pessoa.telefone;
